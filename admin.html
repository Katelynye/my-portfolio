Your portfolio data is stored in a data.json file in a GitHub repository. This admin panel reads and writes that file via the GitHub API using a Personal Access Token.

🤖 AI Screenshot Parsing Free

Uses Google Gemini API (free — 1,500 requests/day, no credit card needed). Get your free key at aistudio.google.com/app/apikey. Key is stored only in your browser.

📖 How to set up GitHub
  1. Create a new public GitHub repository (e.g. my-portfolio)
  2. Upload the data.json file to the repo root
  3. Go to Settings → Pages and enable GitHub Pages from the main branch
  4. Go to Settings → Developer settings → Personal access tokens → Tokens (classic)
  5. Generate a new token with repo scope and paste it above
  6. Your public dashboard URL will be: https://USERNAME.github.io/REPO/
  7. Your data URL will be: https://raw.githubusercontent.com/USERNAME/REPO/main/data.json
